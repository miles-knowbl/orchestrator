# Content Analysis

## Overview
Systematically examine content to identify reusable components, patterns, and actionable insights.

## Process
1. **Content Ingestion**: Parse and understand the source material
2. **Pattern Recognition**: Identify recurring themes, structures, or solutions
3. **Skill Extraction**: Pull out specific procedures or instructions that could be reused
4. **Categorization**: Organize findings by type, phase, and applicability
5. **Validation**: Assess confidence and usefulness of extracted elements

## Output Format
- Structured data with clear categorization
- Confidence scoring for reliability
- Context for when to apply findings
- Reasoning for extraction decisions
