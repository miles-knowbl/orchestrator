# Debrief Template Reference

What to capture in meeting debriefs.

## Capture Checklist

### Meeting Basics

- [ ] Date and time
- [ ] Duration
- [ ] Attendees (their side)
- [ ] Attendees (our side)
- [ ] Meeting type/purpose

### Immediate Captures

Do within 30 minutes of meeting end:

- [ ] Top 3 key takeaways
- [ ] Any surprises
- [ ] Strongest quote from prospect
- [ ] Gut feeling on deal health

### Intelligence Categories

#### Pain Points

- [ ] New pain points mentioned
- [ ] Existing pain confirmed/updated
- [ ] Severity assessments
- [ ] Quantified impacts
- [ ] Direct quotes

**Template:**
```
Pain: [Description]
Severity: [High/Medium/Low]
Quote: "[Direct quote]" — [Speaker]
Impact: [Quantified if available]
```

#### Budget/Timeline

- [ ] Budget mentioned or confirmed
- [ ] Approval status
- [ ] Timeline mentioned or confirmed
- [ ] Deadline drivers
- [ ] Fiscal constraints

**Template:**
```
Budget: [Amount or range]
Status: [Approved/Needs approval/Unknown]
Timeline: [Date or timeframe]
Driver: [What's causing the deadline]
```

#### Stakeholders

- [ ] New stakeholders mentioned
- [ ] Sentiment updates
- [ ] Role clarifications
- [ ] Key quotes from each
- [ ] Concerns raised

**Template:**
```
Stakeholder: [Name]
Title: [Title]
Sentiment: [Supportive/Neutral/Skeptical]
Key Quote: "[Quote]"
Concern: [If any]
Next Step With Them: [Action]
```

#### Technical

- [ ] Stack components mentioned
- [ ] Integration requirements
- [ ] Security/compliance needs
- [ ] Technical concerns raised
- [ ] Data availability

**Template:**
```
System: [Name]
Type: [CRM/Support/Auth/etc.]
Integration: [Required/Nice/Not needed]
Concern: [If any]
```

#### Competitive

- [ ] Competitors mentioned
- [ ] Sentiment about competitors
- [ ] Build vs. buy discussion
- [ ] Status quo considerations

#### Use Case

- [ ] Scope clarified
- [ ] Specific use cases
- [ ] In/out of scope
- [ ] Success criteria
- [ ] Phasing

## Objective Evaluation

### Status Options

| Status | Meaning |
|--------|---------|
| ✓ Achieved | Got clear answer |
| Partial | Some progress |
| Not Achieved | Didn't get answer |
| Deferred | Agreed to discuss later |
| N/A | Not relevant to this meeting |

### For Each Objective

```
Objective: [What we wanted to learn]
Status: [Achieved/Partial/Not Achieved]
Result: [What we learned]
Follow-up: [If partial/not achieved, what's next]
```

## Deal Progression Signals

### Positive Progression Signs

- Agreed to next meeting
- Introduced new stakeholders
- Shared internal information
- Asked about pricing/timeline
- Used "we" language
- Provided access/documentation
- Expressed urgency

### Negative Progression Signs

- Vague about next steps
- New objections raised
- Timeline pushed
- Stakeholders disengaged
- Mentioned competitors positively
- Avoided commitment

### Stage Readiness

Questions to answer:
1. Did we get what we needed at this stage?
2. Are we ready to advance to next stage?
3. What's blocking stage advancement?

## Action Items Template

### Immediate (24 hours)

```
[ ] [Action] — [Owner] — [Due]
```

Typical immediate actions:
- Send follow-up email
- Share requested materials
- Schedule next meeting
- Make promised introduction

### Short-term (This week)

```
[ ] [Action] — [Owner] — [Due]
```

Typical short-term actions:
- Create requested content
- Update deal files
- Brief internal team
- Prepare for next meeting

### Questions to Answer

```
[ ] [Question] — [How to answer]
```

Things we need to find out before next meeting.

## Follow-Up Email Template

```
Subject: [Reference to meeting] — next steps for [Company]

[Name],

Thank you for [specific acknowledgment of meeting].

Key takeaways from our conversation:
- [Takeaway 1]
- [Takeaway 2]
- [Takeaway 3]

Next steps:
1. [Your action item]
2. [Their action item, if any]
3. [Scheduled meeting or proposed date]

[Any materials attached]

[Warm close]

[Signature]
```

---

## Quality Questions

After completing debrief, ask:

1. If someone else read this, would they understand the deal?
2. Did I capture specific numbers, not vague impressions?
3. Do I have direct quotes, not paraphrases?
4. Are action items specific with owners and dates?
5. Did I update all relevant intelligence files?
6. Can I write a great follow-up email from this?

## Timing Guidelines

| Task | Ideal Timing |
|------|--------------|
| Immediate captures | Within 30 min |
| Full debrief | Within 2 hours |
| Follow-up email | Same day |
| Intelligence updates | Same day |
| Score recalculation | Within 24 hours |
