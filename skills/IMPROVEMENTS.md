# Skills Improvement Queue

This file tracks improvement suggestions captured via the `capture_improvement` MCP tool. Improvements are reviewed periodically and applied to skills.

## Pending Improvements

*No pending improvements. Use `improve: [feedback]` or the `capture_improvement` tool to suggest improvements.*

---

## Applied Improvements

*Improvements that have been reviewed and applied to skills.*

---

## How to Contribute

1. **During skill execution**: Say `improve: [your feedback]` to capture improvement ideas
2. **Via MCP tool**: Use `capture_improvement` with feedback, source, category, and skill name
3. **Manual edit**: Add an entry below following the format

### Improvement Entry Format

```markdown
### IMP-{number}: {Title}
- **Date**: YYYY-MM-DD
- **Source**: {project name} {phase}
- **Category**: enhancement|bug|clarification|new-feature
- **Status**: pending|applied
- **Skill**: {skill-name}
- **Feedback**: {detailed feedback}
- **Action**: [To be determined / Applied in version X.Y.Z]
```
