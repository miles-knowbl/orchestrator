# Dimension Definitions

Quick reference for understanding and populating each dimension of the taste schema.

## Textual Dimensions

### 1. Narrative Mechanics
**What it captures:** The structural patterns that give stories their shape and momentum.

| Sub-dimension | Definition | Signal phrases |
|--------------|------------|----------------|
| Arc types | Recurring story structures (Shame→Liberation, Weakness→Strength, Rejection→Fit, Fear→Mastery, Problem→Creation) | "When I was...", "I used to...", "Everything changed when..." |
| Number anchors | Specific quantities that ground abstract claims | Exact numbers (27, not "about 30"), ratios (9x slower), scales (55M users) |
| Sensory details | Physical/environmental details that create visceral memory | Body sensations, environment descriptions, physical actions |
| Proper nouns | Named entities that anchor in verifiable reality | Institutions, places, people, pop culture references |

### 2. Rhetorical Signature
**What it captures:** Distinctive persuasion patterns that make explanations sticky.

| Sub-dimension | Definition | Signal phrases |
|--------------|------------|----------------|
| Explanatory analogies | Domain bridges that make abstract concrete | "It's like...", "Think of it as...", "Imagine..." |
| Humor modes | Strategic humor deployment (self-deprecating, absurdist, observational) | Function: disarm, build trust, create memory, soften |
| Contrarian inversions | Flipping conventional wisdom with lived proof | "Most people think X. I've found Y." |

### 3. Emotional Architecture
**What it captures:** The layered emotional structure that creates resonance and trust.

| Sub-dimension | Definition | Formula |
|--------------|------------|---------|
| Vulnerability layer | Specific + past + overcome admissions | Shame/fear/failure that is documented and resolved |
| Triumph layer | Earned victories paired with documented struggle | Improbability quantified, present-day proof offered |
| Gratitude layer | Specific thanks with named targets and impacts | "Grateful to [person] for [action] because [impact]" |

### 4. Identity Threads
**What it captures:** Consistent identity claims forming the narrative persona.

| Sub-dimension | Definition | Strength hierarchy |
|--------------|------------|-------------------|
| Core claims | "I am", "I do", "I believe" statements | "I do" > "I am" > "I believe" (action-based strongest) |
| Belonging dynamics | Outsider→insider transitions | Outsider marker → belonging moment → bridge identity |

### 5. Teaching Moves
**What it captures:** Pedagogical patterns that make advice sticky and actionable.

| Sub-dimension | Definition | Naming patterns |
|--------------|------------|-----------------|
| Framework delivery | Packaged wisdom with memorable names | Numbers ("5 Pillars"), alliteration, metaphor, pop culture |
| Advice inversions | Common advice explicitly flipped with proof | "People say X. I say Y. Here's why: [story]" |

### 6. Audience Relationship
**What it captures:** How connection and permission are created.

| Sub-dimension | Definition | Signal |
|--------------|------------|--------|
| Universal hooks | Entry points most listeners recognize | Internal monologue language, early placement |
| Parasocial invitations | Private moments shared to create intimacy | Personal rituals, family moments, spiritual practices |
| Permission grants | What audience is allowed to feel/do/be | Implicit or explicit, tied to story resolution |

### 7. Narrative Metadata
**What it captures:** Context-dependent deployment patterns.

| Sub-dimension | Definition | Elements |
|--------------|------------|----------|
| Audience adaptations | How content shifts for different audiences | Story selection, tone, credibility markers, vocabulary |
| Story clustering | How stories link into sequences | Chains that build narrative momentum |
| Signature phrases | Recurring language patterns | Openers, transitions, closers |

## Visual Dimensions

### 1. Color System
**What it captures:** The emotional and aesthetic role of color.

| Element | Questions to ask |
|---------|-----------------|
| Primary colors | What 2-3 colors dominate? What emotions do they carry? |
| Secondary colors | What colors support the primaries? |
| Accent colors | What colors create emphasis/highlights? |
| Forbidden colors | What colors never appear? |
| Relationships | Complementary? Analogous? High contrast? Muted? |

### 2. Typography System
**What it captures:** Type choices and their personality.

| Element | Questions to ask |
|---------|-----------------|
| Headings | Bold/light? Serif/sans? Personality conveyed? |
| Body | Readability priority? Density? |
| Accents | Special treatment for quotes/callouts? |
| Hierarchy | How do sizes/weights create structure? |

### 3. Composition System
**What it captures:** How elements are arranged in space.

| Element | Questions to ask |
|---------|-----------------|
| Layout | Grid-based? Asymmetric? Centered? |
| Whitespace | Dense or airy? What does space communicate? |
| Flow | How does the eye move through content? |
| Aspect ratios | Common formats (16:9, 1:1, 9:16)? |

### 4. Imagery System
**What it captures:** Visual content style and vocabulary.

| Element | Questions to ask |
|---------|-----------------|
| Illustration | Flat? Detailed? Hand-drawn? 3D? |
| Icons | Line weight? Filled or outlined? Simple or complex? |
| Motifs | What visual elements recur? |
| Metaphors | How are abstract concepts visualized? |
| Photography | Candid/staged? Filtered? Subject matter? |

### 5. Mood/Atmosphere
**What it captures:** The overall feeling evoked.

| Element | Questions to ask |
|---------|-----------------|
| Energy | Low/medium/high? Calm or dynamic? |
| Texture | Clean digital? Subtle grain? Heavy texture? |
| Lighting | Bright? Moody? Natural? Dramatic? |
| Emotion | What feelings should visuals evoke? |

## Confidence Calibration Guide

| Level | Text sources | Image sources | Pattern consistency |
|-------|-------------|---------------|-------------------|
| High | 20+ | 10+ | 70%+ |
| Medium | 5-20 | 3-10 | 40-70% |
| Low | <5 | <3 | <40% |

For **low-confidence** dimensions:
- Still populate all fields
- Use phrases like "extrapolated from limited data", "inferred", "pattern unclear"
- Note which specific sub-dimensions have insufficient evidence
