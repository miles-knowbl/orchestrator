# Proposal Builder

Create compelling, professional proposals.

## When to Use

- After priorities are established
- When context is fully cultivated
- To generate client-ready documents
- When you say: "create the proposal", "generate the document", "write this up"

## Required Deliverables

| Deliverable | Location | Description |
|-------------|----------|-------------|
| PROPOSAL.md | Root | Complete proposal document |

## Process

```
┌─────────────────────────────────────────────────────────────┐
│                  PROPOSAL BUILDER                           │
│                                                             │

│                                                             │

│                                                             │
```

## Proposal Sections

| Section | Purpose |
|---------|---------|
| Executive Summary | Hook and overview |
| Problem Statement | Show understanding |
| Proposed Solution | The approach |
| Scope | Boundaries and deliverables |
| Timeline | When things happen |
| Investment | Cost and value |
| Why Us | Credibility |
| Next Steps | Call to action |

## Templates

- **Standard**: Full proposal with all sections
- **Brief**: Executive summary + scope + investment
- **Technical**: Detailed technical approach

## Success Criteria

- All claims evidenced from context
- Clear value proposition
- Professional formatting
- Actionable next steps
